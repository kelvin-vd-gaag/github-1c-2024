1. Maak een github account aan
  2. Download de software gitkraken via https://www.gitkraken.com/
  3. Installeer de software gitkraken en log in op gitkraken met je github account
  4. Initialiseer een GIT repository op je lokale folder (dat mag een bestaande website zijn of een nieuwe)
  5. Maak wat aanpassingen en sla deze op als een commit
  6. Push je commits naar je github profiel via gitkraken
